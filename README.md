# Walmart-Stores-Analysis

In this project we took THree dats sets and did a analysis and found out the following:
1. Higher the size more is the sales.
2. Each year almost same pattern so demand can be planned accordingly by considering last year's patterns.
3. End year is having higher sales. 
4. Store 20 having highest average sales. 
5. Store 5 is having lowerst sales 5 dept 92 is having highest sales. 
6. Dept 47 is having lowest sales. 
7. Type a belongs to the larger stores. 
8. Type c belongs to the smaller stores.
